# Sallehli

This is a mobile application for managing service requests with a technician matching system. The app allows customers to book services, track their orders, and communicate with technicians.

## Features
- User Authentication
- Order Tracking
- Technician List
- Review System
- Admin Dashboard

## Getting Started
1. Clone the repository.
2. Run `flutter pub get` to install dependencies.
3. Configure Firebase settings in `firebase_config.json`.
